<h3>Flow Description</h3>
<ul>
<li><strong>Application Entry:</strong> User opens the application and the Header component loads with logo, search, navigation tabs, and user actions</li>
<li><strong>Navigation:</strong> Users can navigate through different sections using header tabs or sidebar menu items</li>
<li><strong>Dashboard Interaction:</strong> Main dashboard displays cards with various content, headers, and action buttons</li>
<li><strong>Sidebar Navigation:</strong> Collapsible sidebar provides additional navigation options with icon-only or full menu views</li>
<li><strong>User Actions:</strong> Header provides access to profile, notifications, settings, and theme toggle</li>
<li><strong>Card Interactions:</strong> Cards support various actions like save, cancel, and other primary/secondary actions</li>
<li><strong>Search Functionality:</strong> Global search filters and displays relevant content</li>
<li><strong>Responsive Design:</strong> Components adapt to different screen sizes and user preferences</li>
</ul>
