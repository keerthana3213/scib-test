### 4.1 HeaderComponent

Component Type: Layout

Mapped From A1: HeaderComponent (aava-header)

Purpose: Application-wide navigation and branding with logo, navigation tabs, search, and user profile

Parent: Application

Children:
- LogoSection
- NavigationTabs (aava-tabs)
- SearchComponent
- UserProfileSection (aava-avatar)

Dependencies:
- @aava/play-core (AavaHeaderComponent)
- @aava/play-core (AavaTabsComponent)
- @aava/play-core (AavaIconComponent)
- @aava/play-core (AavaAvatarComponent)

Library Components Used:
- aava-header → Main header container
- aava-tabs → Navigation tabs
- aava-icon → Search and action icons
- aava-avatar → User profile avatar

TypeScript Specification

Inputs:
- logoUrl: string
- navigationItems: NavigationItem[]
- userProfile: UserProfile
- searchEnabled: boolean
- activeTabId: string

Outputs:
- onNavigationChange: (tabId: string) => void
- onSearch: (query: string) => void
- onUserAction: (action: string) => void

State:
- activeTab: string
- searchQuery: string
- isUserMenuOpen: boolean

Methods:

- onSearchChange() → Handles search input

- onTranslateIconClick() → Language toggle handler
- onDayLightModeIconClick() → Settings menu handler
- getLoggedInUserLogo() → Returns user avatar URL

<aava-header containerClass="app-header">

    <div class="logo">

  <div center-content class="header__center tabs-header">

      [tabs]="navigationTabs"

- display: grid with three-column layout
- grid-template-columns: auto 1fr auto
- align-items: center

Spacing:
- padding: 16px 24px
- gap: 16px between sections

Component Styles:
- .header__left: flex layout for logo
- .header__center: flex layout for navigation
- .header__right: flex layout for actions
- .search-box: flex with icon and input
- .icon: cursor pointer with hover effects

Responsive:

User Action → UI → API

Component Type: Layout

        <span class="user-name">{{ userProfile.name }}</span>
        <span class="user-role">{{ userProfile.role }}</span>
      </div>
    </div>
  </div>

Layout:

- width: 280px (expanded), 64px (collapsed)

Component Styles:
- .sidebar-header: flex with collapse toggle

- .nav-link: flex layout with icon and text
- .sidebar-footer: user profile section

INTERACTION FLOW

User Action → UI → API
- Click menu item → update selection → navigate to page → load page data

- Sort column → update sort state → reload data → refresh table
- Filter data → apply filters → reload data → update display
- Edit row → open edit modal → save changes → refresh row
- Delete row → show confirmation → delete record → remove from table
- Page change → update pagination → load new page → refresh display
