Jira Requirement Summary

Story: Parent can allocate and manage funds for a youth account

Description:
AS a parent or guardian I WANT to allocate and manage funds for my child’s youth account SO THAT I can teach financial responsibility while maintaining visibility and control over spending.

Acceptance Criteria:
1. Access youth account dashboard: Parent sees dashboard with current balance and recent activity.
2. Allocate funds: Parent can transfer funds from their primary account to the youth account.
3. Set spending limit: Parent can configure a weekly spending limit, which is displayed on the dashboard.
4. View youth spending activity: Parent can view recent transactions from the youth account.
5. Handle insufficient balance: If parent’s account lacks funds, an error message is shown.

UI Tasks (from subtasks):
- Design youth account dashboard layout
- Design fund transfer interaction for youth account
- Design spending limit configuration interface
- Design youth account activity view

Component Responsibilities

HeaderComponent
- Displays logo, navigation tabs, and user avatar.
- Handles navigation and user menu actions.

SearchAndFilterBar
- Allows searching and filtering of dashboard data (role, group, initiator, status).
- Triggers filter actions and clearing of filters.

BalanceCard
- Shows current youth account balance and configured spending limit.
- Displays parent/youth avatar.

ActivitySection
- Tabbed navigation for recent activity, fund transfers, and spending limits.
- Displays a list of activity items or an empty state if none exist.

ActivityItemCard
- Shows details of a single transaction or activity (date, amount, type, status).

Snackbar
- Displays error or success messages (e.g., insufficient funds).

Dialog/Modal
- Used for Add Funds and Set Spending Limit interactions.
- Contains forms for entering amounts, selecting accounts, and confirming actions.
