1. Jira Requirement Summary

Story: Parent can allocate and manage funds for a youth account

Description:
AS a parent or guardian I WANT to allocate and manage funds for my child’s youth account SO THAT I can teach financial responsibility while maintaining visibility and control over spending

Acceptance Criteria:
- Scenario 1: Access youth account dashboard (dashboard with current balance and recent activity)
- Scenario 2: Allocate funds to youth account (Add Funds)
- Scenario 3: Set spending limit (configure weekly spending limit, display on dashboard)
- Scenario 4: View youth spending activity (recent transactions)
- Scenario 5: Handle insufficient balance during fund allocation (error message)

UI Tasks / Subtasks:
- Design youth account dashboard layout
- Design fund transfer interaction for youth account
- Design spending limit configuration interface
- Design youth account activity view

5. Component Responsibilities

- YouthAccountDashboardPage: Orchestrates layout, data flow, and state for the dashboard screen.
- HeaderComponent: Displays app logo, search bar, profile avatar, and main action buttons.
- SidebarComponent: Provides navigation for dashboard, activity history, fund transfer, and spending limit.
- AccountOverviewCard: Shows account holder, balance, spending limit, and quick actions.
- RecentActivityCard: Lists recent transactions (debits/credits, merchant, amount, status).
- QuickGuidanceCard: Shows tips and guidance for parents.
- SnackbarComponent: Displays feedback messages (e.g., insufficient funds error).
- TransactionItem: Represents a single transaction (merchant, amount, type, status).

  -> SidebarMenuItems (menuItems, selectedMenu)

  -> (message, type)
