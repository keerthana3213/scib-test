1. Jira Requirement Summary

Story: Parent can allocate and manage funds for a youth account

Description:
AS a parent or guardian I WANT to allocate and manage funds for my child’s youth account SO THAT I can teach financial responsibility while maintaining visibility and control over spending

Acceptance Criteria:
- Scenario 1: Access youth account dashboard (dashboard with current balance and recent activity)
- Scenario 2: Allocate funds to youth account (Add Funds, transfer from parent to youth)
- Scenario 3: Set spending limit (configure weekly limit, display on dashboard)
- Scenario 4: View youth spending activity (recent transactions)
- Scenario 5: Handle insufficient balance during fund allocation (error message)

UI Tasks (from subtasks):
- Design youth account dashboard layout
- Design fund transfer interaction for youth account
- Design spending limit configuration interface
- Design youth account activity view

5. Component Responsibilities

- YouthAccountDashboardPage: Main container, manages state and data flow, coordinates dialogs and feedback.
- HeaderComponent: Displays app branding, navigation, user avatar.
- SidebarComponent: Main navigation for parent portal.
- AccountOverviewCard: Shows youth’s name, avatar, current balance, and guidance tips.
- ActionButtonsContainer: Hosts Add Funds and Set Limit actions.
- RecentActivityTimeline: Displays recent transactions (uses Timeline component).
- GuidanceTipsSection: Shows financial tips or info for parents.
- AddFundsDialog: Modal for adding funds, handles form, validation, and error state.
- SetLimitDialog: Modal for configuring spending limit, handles form and success state.
- Snackbar/Toast: Shows feedback for success/error actions.


6. Data Flow / Props

YouthAccountDashboardPage
  -> HeaderComponent (userData)
  -> SidebarComponent (navigationItems)
  -> AccountOverviewCard (youthData, balance, guidanceTips)
  -> RecentActivityTimeline (activityList)
  -> AddFundsDialog (open, onSubmit, onError)
  -> SetLimitDialog (open, onSubmit, onSuccess)
  -> Snackbar/Toast (message, type)

AccountOverviewCard
  -> AvatarComponent (avatarUrl)
  -> YouthNameTag (youthName)
  -> CurrentBalanceDisplay (balance)
  -> GuidanceTipsSection (tips)
  -> ActionButtonsContainer (onAddFunds, onSetLimit)

AddFundsDialog
  -> AddFundsForm (amount, onSubmit)
  -> ErrorSnackbar (visible on insufficient funds)

SetLimitDialog
  -> SetLimitForm (limit, onSubmit)
  -> SuccessSnackbar (visible on success)
