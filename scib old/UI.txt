1. Jira Requirement Summary

Story: Parent can allocate and manage funds for a youth account

Description:
AS a parent or guardian I WANT to allocate and manage funds for my child’s youth account SO THAT I can teach financial responsibility while maintaining visibility and control over spending

Acceptance Criteria:
- Scenario 1: Access youth account dashboard (dashboard with current balance and recent activity)
- Scenario 2: Allocate funds to youth account (Add Funds action, transfer from parent to youth)
- Scenario 3: Set spending limit (configure weekly spending limit, display on dashboard)
- Scenario 4: View youth spending activity (recent transactions)
- Scenario 5: Handle insufficient balance (error message on insufficient funds)

UI Tasks / Subtasks:
- Design youth account dashboard layout
- Design fund transfer interaction for youth account
- Design spending limit configuration interface
- Design youth account activity view

5. Component Responsibilities

- HeaderComponent: Displays the application header and navigation tabs.
- TabsComponent: Renders navigation options (HOME, ASK, RESOURCES, REPORTS, ADMIN).
- MainContentContainer: Contains the main dashboard content.
- TitleSection: Shows the "Select ASK type" prompt.
- AskTypeOptionsCard: Container for the ASK type selection buttons.
- OptionButton: Allows user to select "Core" or "Rotational" ASK type.
- ActionButtonsRow: Contains EXIT and START NOW action buttons.
- ExitButton: Triggers exit/cancel flow.
- StartNowButton: Initiates the ASK process (disabled until selection).
- DialogComponent: Handles confirmation, errors (e.g., insufficient funds), and other modal dialogs.
