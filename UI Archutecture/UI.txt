Jira Requirement Summary:

Story: Parent can allocate and manage funds for a youth account
Description: AS a parent or guardian I WANT to allocate and manage funds for my child’s youth account SO THAT I can teach financial responsibility while maintaining visibility and control over spending

Acceptance Criteria:
1. Access youth account dashboard: Parent logs in, navigates to youth account section, sees dashboard with balance and recent activity.
2. Allocate funds: Parent selects 'Add Funds', transfers funds from primary to youth account.
3. Set spending limit: Parent configures weekly spending limit, system saves/displays limit.
4. View youth spending activity: Parent selects activity section, sees recent transactions.
5. Handle insufficient balance: Parent attempts transfer, system shows error if insufficient funds.

Subtasks/UI Tasks:
- Design youth account dashboard layout
- Design fund transfer interaction for youth account
- Design spending limit configuration interface
- Design youth account activity view

Linked Issues:
- Create API to retrieve youth account dashboard details
- Create API for parent to transfer funds to youth account
- Create API for configuring youth spending limit
- Create API to retrieve youth account transaction history
- Define OpenAPI/Swagger specification for youth account APIs

Component Responsibilities:

HeaderComponent:
- Displays navigation bar with menu options and user avatar.
- Handles navigation tab changes.

SidebarComponent:
- Provides navigation menu for other sections (Resources, Reports, Admin).

MainContentContainer:
- Displays dashboard title and option cards.
- OptionCardContainer: Shows "Core" and "Rotational" selection buttons.
- ActionButtonContainer: Shows "EXIT" and "START NOW" buttons.

SnackbarContainer:
- Displays transient notifications (e.g., error for insufficient funds).
