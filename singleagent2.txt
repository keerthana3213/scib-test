Component Type:
Layout

Mapped From A1:
HeaderComponent

Purpose:
Displays search bar, notification icon, avatar chip with parent name

Parent:
YouthAccountManagementPage

Children:
- SearchBar
- Avatar
- ChipTag
- Button (notification)

Dependencies:
- @aava/play-core: AavaHeaderComponent, AavaTabsComponent, AavaIconComponent, AavaAvatarsComponent

Library Components Used:
- Header → main layout
- Tabs → navigation (if needed)
- Search-Bar → search input
- Avatar → parent avatar
- Chip-Tag → parent name
- Button/Icon → notification

--------------------------------------------------
TypeScript Specification
--------------------------------------------------

Inputs:
- searchText: string
- parentName: string
- avatarUrl: string

Outputs:
- onSearchChange(search: string)
- onNotificationClick()
- onAvatarClick()

State:
- searchText: string

Methods:
- handleSearchChange() → emits onSearchChange
- handleNotificationClick() → emits onNotificationClick
- handleAvatarClick() → emits onAvatarClick

--------------------------------------------------
HTML STRUCTURE (PSEUDO CODE ONLY)
--------------------------------------------------

<AavaHeaderComponent>
  <div left-content>
    <img src="{logoUrl}" alt="Brand Logo" />
  </div>
  <div center-content>
    <AavaSearchBar [(ngModel)]="searchText" (input)="handleSearchChange()" placeholder="Search..." />
  </div>
  <div right-content>
    <AavaButton iconName="bell" (userClick)="handleNotificationClick()" />
    <AavaChipTag [label]="parentName" />
    <AavaAvatar [imageUrl]="avatarUrl" (userClick)="handleAvatarClick()" />

CSS SPECIFICATION
--------------------------------------------------

- CSS Grid for three-column header
- Flex for right-content alignment

Spacing:
- Padding: 24px
- Gap: 16px between actions

Component Styles:

Responsive:

- @media (max-width: 768px): left-content collapses to menu icon

--------------------------------------------------
API INTEGRATION
--------------------------------------------------

- None

--------------------------------------------------
ERROR HANDLING
--------------------------------------------------

- None

--------------------------------------------------
INTERACTION FLOW
--------------------------------------------------

### 4.7 FundTransferForm

Component Type:
Feature

Mapped From A1:
FundTransferForm

Purpose:
Form for parent to transfer funds to youth account

Parent:
FundTransferPage

Children:
- Card
- Dropdown (source account)
- Textbox (amount)
- ChipTag (frequent templates)
- Badge (transfer limit)
- Button (transfer/cancel)

Dependencies:
- @aava/play-core: AavaDefaultCardComponent, AavaCardHeaderComponent, AavaCardContentComponent, AavaDropdownComponent, AavaTextboxComponent, AavaChipTagComponent, AavaBadgeComponent, AavaButtonComponent

Library Components Used:
- Card
- Dropdown
- Textbox
- ChipTag
- Badge
- Button

--------------------------------------------------
TypeScript Specification
--------------------------------------------------

Inputs:
- sourceAccounts: string[]
- transferLimit: number
- frequentTemplates: {label: string, amount: number}[]

Outputs:
- onTransferSubmit(payload: {source: string, amount: number})
- onCancel()

State:
- selectedSource: string
- amount: number

Methods:
- handleTransferSubmit() → emits onTransferSubmit
- handleCancel() → emits onCancel
- handleTemplateClick(template) → sets amount

--------------------------------------------------
HTML STRUCTURE (PSEUDO CODE ONLY)
--------------------------------------------------

<AavaDefaultCard>
  <AavaCardHeader>
    <span>Add Funds</span>
    <AavaBadge [label]="'Limit: ' + transferLimit" />
  </AavaCardHeader>
  <AavaCardContent>
    <AavaDropdown [options]="sourceAccounts" [(ngModel)]="selectedSource" placeholder="Select Source Account" />
    <AavaTextbox [(ngModel)]="amount" placeholder="Amount" type="number" />
    <div class="frequent-templates">
      <AavaChipTag *ngFor="let template of frequentTemplates" [label]="template.label" (userClick)="handleTemplateClick(template)" />
    </div>
    <div class="form-actions">
      <AavaButton label="Transfer" variant="primary" (userClick)="handleTransferSubmit()" />
      <AavaButton label="Cancel" variant="secondary" (userClick)="handleCancel()" />
    </div>
  </AavaCardContent>
</AavaDefaultCard>

--------------------------------------------------
CSS SPECIFICATION
--------------------------------------------------

Layout:
- Card: border-radius 24px, box-shadow
- Form: flex column

Spacing:
- Padding: 24px
- Gap: 16px between fields

Component Styles:
- .frequent-templates: display flex, gap 8px
- .form-actions: display flex, gap 12px

Responsive:
- Card width 100% on mobile

--------------------------------------------------
API INTEGRATION
--------------------------------------------------

- POST /api/youth-accounts/{youthAccountId}/fund-transfer → submit transfer

--------------------------------------------------
ERROR HANDLING
--------------------------------------------------

- API failure → handled by parent (SnackbarAlert)
- Validation error → inline message

--------------------------------------------------
INTERACTION FLOW
--------------------------------------------------

User fills form → clicks Transfer → handleTransferSubmit → onTransferSubmit event → API call
User clicks Cancel → handleCancel → onCancel event
User clicks template → handleTemplateClick → sets amount
